# DUET 網站重設計 - 設計決策文件

## 專案目標

重新設計 DUET 珠寶品牌網站，使首頁（index.html）具備高級國際精品的視覺質感，並調整設計工作室頁面（design-studio.html）使兩頁面風格統一，同時保留金工工作室的手工藝氛圍。

---

## 核心設計策略

### 1. 視覺語言系統

#### 色彩系統
- **背景色**：深灰藍（oklch(0.15 0.02 260)）
  - 傳達高級、專業、沉靜的氛圍
  - 避免純黑，增加溫暖感
  
- **文字色**：象牙白（oklch(0.95 0.005 65)）
  - 提供最佳可讀性
  - 傳達優雅、精緻的品牌形象
  
- **強調色**：古銅金色（oklch(0.65 0.15 65)）
  - 象徵工藝精神和奢華
  - 用於 CTA 按鈕、重點文字和互動元素

#### 排版系統
- **標題字體**：Playfair Display（serif）
  - 高級感、優雅、傳統工藝的象徵
  - 用於所有 h1-h6 和 .font-display 類
  
- **正文字體**：Noto Serif TC（serif）
  - 易讀性強、現代感
  - 支援繁體中文，保持品牌本地化
  
- **手工字體**：Caveat（cursive）
  - 用於特殊強調，傳達手工藝感
  - 用於 .font-handcraft 類

### 2. 設計哲學

#### 首頁（index.html）
**目標**：展現品牌故事和高級感，避免傳統電商風格

**實現方式**：
- 英雄背景圖片：展示珠寶工作室的精美光感
- 非對稱排版：左文右圖的設計，避免單調的居中佈局
- 充足留白：每個區塊都有清晰的視覺分隔
- 故事導向：強調「每件珠寶都是一個故事」的品牌理念
- 流程透明化：清晰展示從想法到完成的 4 個步驟

**關鍵區塊**：
1. 導航欄：固定頂部，深色背景 + 毛玻璃效果
2. 英雄區：全屏背景圖 + 視差滾動效果
3. 品牌故事：左文右圖，展示工藝細節
4. 設計流程：4 步驟卡片式設計
5. 客戶故事：3 個真實案例展示
6. FAQ：可展開的問題列表
7. CTA：強調行動呼籲
8. 頁腳：品牌信息和導航

#### 設計工作室（design-studio.html）
**目標**：升級為「工作台」風格，提升手工藝感，避免冰冷的參數面板

**實現方式**：
- 卡片式設計：每個選項都是獨立的卡片，提升視覺層級
- 實時預覽：3D 珠寶預覽，展示材質、電鍍和尺寸的變化
- 價格實時計算：每個選擇都有視覺反饋和價格更新
- 儀式感：強調手工製作的過程和品質
- 工作台氛圍：深色背景 + 卡片邊框，模擬真實工作環境

**關鍵功能**：
1. 3D 預覽區：展示珠寶設計，支援多角度查看
2. 字母選擇：下拉菜單 + 字體選擇按鈕
3. 材質選擇：925 銀 vs 黃銅
4. 電鍍選擇：無、18K 金、18K 白K金、18K 玫瑰金
5. 表面質地：亮面 vs 霧面
6. 尺寸選擇：8-18mm 多個選項
7. 價格顯示：實時計算，突出強調
8. 行動按鈕：加入購物車、保存設計、重新開始 AI 諮詢

## 實現細節

### 1. 全局樣式更新（client/src/index.css）

```css
/* 新增 Google Fonts 導入 */
@import url('https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;500;600;700&family=Caveat:wght@400;700&display=swap');

/* 更新 CSS 變數 */
:root {
  --primary: oklch(0.65 0.15 65); /* 古銅金色 */
  --background: oklch(0.15 0.02 260); /* 深灰藍 */
  --foreground: oklch(0.95 0.005 65); /* 象牙白 */
  --accent: oklch(0.65 0.15 65); /* 古銅金色 */
}

/* 新增字體類 */
.font-display { font-family: 'Playfair Display', serif; }
.font-handcraft { font-family: 'Caveat', cursive; }
```

### 2. 首頁組件（client/src/pages/Home.tsx）

**結構**：
- 固定導航欄（sticky）
- 英雄區（全屏背景圖 + 視差效果）
- 品牌故事區（左文右圖）
- 設計流程區（4 步驟卡片）
- 客戶故事區（3 個案例）
- FAQ 區（可展開列表）
- CTA 區（強調行動）
- 頁腳

**特色**：
- 使用 Unsplash 和生成的高質量圖片
- 視差滾動效果（`transform: translateY(${scrollY * 0.5}px)`)
- 動畫進入效果（@keyframes fadeIn）
- 響應式設計（md: 斷點）

### 3. 設計工作室組件（client/src/pages/DesignStudio.tsx）

**結構**：
- 固定頭部（保存設計、加入購物車按鈕）
- 左側：3D 預覽 + 設計描述
- 右側：價格顯示 + 自定義面板

**特色**：
- 實時價格計算
- 3D 珠寶預覽（模擬圓形設計）
- 卡片式控制面板
- 粘性側邊欄（sticky bottom）
- 響應式網格（lg:grid-cols-3）

### 4. 路由配置（client/src/App.tsx）

```tsx
<Route path="/" component={Home} />
<Route path="/design-studio" component={DesignStudio} />
```

- ✅ 深灰藍 + 古銅金色：高級、專業
- ✅ 非對稱排版：視覺豐富、動態
- ✅ 故事導向：情感連接
- ✅ 單一高質量背景：避免重複，保持工作室氛圍

### 原設計工作室問題
- ❌ 冰冷的參數面板：缺乏手工藝感
- ❌ 平面設計：視覺層級不足
- ❌ 缺乏實時反饋：用戶體驗不佳

### 新設計工作室改進

## 技術實現

### 使用的技術棧
- React 19 + TypeScript
- Tailwind CSS 4（OKLCH 色彩空間）
- Wouter（路由）
- Lucide React（圖標）
- shadcn/ui（UI 組件）

### 關鍵 CSS 特性
- CSS 變數（主題系統）
- Grid 和 Flexbox（響應式佈局）
- 視差滾動（transform）
- 毛玻璃效果（backdrop-blur）
- 過渡和動畫（transition、@keyframes）

### 響應式斷點
- Mobile：默認
- Tablet：md (768px)
- Desktop：lg (1024px)
