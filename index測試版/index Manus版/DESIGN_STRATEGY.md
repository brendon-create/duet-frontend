# DUET 網站重設計 - 設計策略文件

## 設計哲學：「工藝與優雅的交織」

DUET 是一個客製化珠寶品牌，強調每件作品都是獨特的故事。我們的設計需要傳達三個核心價值：
1. **工藝精神**：手工製作、細節考究、金工工作室的專業感
2. **高級感**：與國際頂級珠寶品牌並肩的視覺質感
3. **個性化**：使用者在設計過程中感受到「親手創作」的儀式感

---

## 視覺語言系統

### 配色方案

| 元素 | 顏色 | OKLCH | 用途 |
|------|------|-------|------|
| 主背景 | 深灰藍 | oklch(0.15 0.02 260) | 頁面背景，營造高級感 |
| 文字主色 | 象牙白 | oklch(0.95 0.005 65) | 正文、標題 |
| 強調色 | 古銅金 | oklch(0.65 0.15 65) | CTA 按鈕、裝飾、邊框 |
| 次強調 | 淡金 | oklch(0.75 0.08 70) | 懸停效果、次級按鈕 |
| 分隔線 | 深灰 | oklch(0.25 0.01 260) | 邊框、分隔線 |
| 背景層 | 深灰黑 | oklch(0.12 0.01 260) | 卡片背景、深層次 |

**配色邏輯**：深色背景強調產品和文字，古銅金色象徵工藝和奢華，避免使用過於飽和的顏色。

### 排版系統

**標題字體**：Playfair Display（serif，高級感）
- 超大標題（H1）：72px，字重 700，行高 1.1
- 大標題（H2）：48px，字重 600，行高 1.2
- 中標題（H3）：32px，字重 600，行高 1.3

**正文字體**：Noto Serif TC（serif，易讀）
- 正文：16px，字重 400，行高 1.6，字間距 0.5px
- 小文字：14px，字重 400，行高 1.5

**裝飾字體**：Caveat（手寫體，工藝感）
- 用於特殊標籤、簽名式文字，傳達手工感

### 間距系統

- **基礎單位**：8px
- **主要間距**：16px, 24px, 32px, 48px, 64px
- **頁面邊距**：左右 32px（桌面）、16px（平板）、16px（手機）
- **卡片內邊距**：24px
- **元素間距**：16px（同級元素）、8px（相關元素）

### 陰影系統

- **淺陰影**：0 2px 8px rgba(0, 0, 0, 0.3)
- **中陰影**：0 8px 24px rgba(0, 0, 0, 0.4)
- **深陰影**：0 16px 40px rgba(0, 0, 0, 0.5)
- **內陰影**：inset 0 2px 4px rgba(255, 255, 255, 0.1)

## 頁面結構設計

### Index.html - 品牌故事頁

**目標**：展現 DUET 的品牌故事、工藝精神和高級感，引導使用者進入設計工作室。

**結構**：
1. **導航欄**（固定）：深灰藍背景，DUET 標誌 + 導航菜單 + CTA 按鈕
2. **英雄區**：全寬背景圖片（珠寶工作室或精美珠寶），疊加深色漸變，文案居左
3. **品牌故事區**：左文右圖，講述 DUET 的故事
4. **工藝流程區**：四個步驟卡片，展示客製化過程
5. **客戶故事區**：輪播展示客戶作品和故事
6. **常見問題區**：可展開的 FAQ 卡片
7. **CTA 區**：強調「開始設計」的號召
8. **頁腳**：聯絡方式、社群媒體、法律信息

**視覺特點**：
- 避免重複背景圖片，使用單一高質量圖片
- 非對稱排版，左右交替
- 充足的留白，每個區塊之間有明確的視覺分隔
- 平滑的滾動動畫，但不過度

### Design-studio.html - 設計工作室

**目標**：保留工作室感，但升級視覺質感，讓使用者感受到「親手設計珠寶」的儀式感。

**改進方向**：
1. **背景**：保持深色，但添加微妙的紋理或光感
2. **參數面板**：從冰冷的數位風格升級為「工作台」風格
   - 使用卡片式設計，而非平面列表
   - 添加視覺反饋（如滑塊移動時的光感變化）
   - 使用工藝相關的圖標和文案
3. **3D 預覽**：強調珠寶的光感和質感
4. **保存和購買**：使用古銅金色按鈕，強調行動的重要性

## 互動設計

### 微交互

- **按鈕懸停**：背景色變淡，文字變亮，陰影增加
- **卡片懸停**：輕微上升（transform: translateY(-4px)），陰影加深
- **滑塊移動**：實時更新預覽，伴隨輕微的光感變化
- **頁面滾動**：平滑過渡，不突兀

### 動畫

- **進入動畫**：元素從下方淡入，持續 0.6s，使用 ease-out 時間函數
- **懸停動畫**：0.2s 的平滑過渡
- **加載動畫**：旋轉的金色圓環，象徵工藝過程

## 設計決策

### 為什麼選擇深色背景？
1. 深色背景能突出珠寶的光感和質感
2. 符合高級珠寶品牌的視覺語言
3. 與現有 design-studio.html 的風格一致

### 為什麼選擇古銅金色？
1. 象徵金工工藝和奢華
2. 在深色背景上有很好的對比度
3. 與珠寶製作的材料相呼應

### 為什麼使用 Playfair Display + Noto Serif TC？
1. Playfair Display 是高級珠寶品牌常用的字體，傳達優雅感
2. Noto Serif TC 提供清晰的中文支持，易於閱讀
3. Serif 字體整體傳達「經典」和「工藝」的感覺

### 為什麼避免重複背景圖片？
1. 重複背景會分散注意力，削弱高級感
2. 單一高質量圖片更能突出內容
3. 符合國際頂級珠寶品牌的設計慣例
